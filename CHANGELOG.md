## Versioning Scheme

- **Major version (X.0.0)**: Breaking changes to API or core architecture
- **Minor version (0.X.0)**: New features, enhancements, or research breakthroughs
- **Patch version (0.0.X)**: Bug fixes, documentation updates, minor improvements

## AERI Developmental Stages

This changelog also tracks AERI's cognitive development milestones:

### Stage 0: Pre-linguistic
- Character pattern recognition
- Basic segmentation learning

### Stage 1: Lexical Acquisition
- First words learned
- Vocabulary building
- Word-meaning associations

### Stage 2: Early Grammar
- Two-word combinations
- Basic sentence structures
- Question formation

### Stage 3: Dialogic Competence
- Conversational turn-taking
- Context maintenance
- Emotional expression

### Stage 4: Advanced Reasoning
- Abstract concepts
- Complex sentence structures
- Creative language use

## How to Update This Changelog

### For Contributors
Add your changes to the [Unreleased] section following the format:

\```
### Added
- [Brief description of new feature]

### Changed
- [Description of changes to existing functionality]

### Fixed
- [Description of bug fixes]
\```

### For Maintainers
When releasing a new version:

1. Update the version number in `pyproject.toml`
2. Move [Unreleased] changes to new version section
3. Update the date
4. Create a new [Unreleased] section
5. Create a Git tag for the release
